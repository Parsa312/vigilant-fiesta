---
title: "Use a model API"
description: "Make a request to a model that Lightning AI hosts."
---

Lightning AI hosts models you can call directly with the API. You can call these models from within your application.

## Before you begin

Retrieve your API key from the Lightning AI dashboard. To retrieve your key:

1. Select the icon for your account in the top-left corner.
2. Select **Keys**.
3. Select the icon next to the key to copy it.

To make requests, install the litai Python SDK — Lightning's Python library for model inference:

```bash
pip install litai
```


After you've installed the library, you can use the following code to make a request.

```python
from litai import LLM
llm = LLM(model="openai/gpt-5", api_key="<LIGHTNING_API_KEY>")
print(llm.chat("Hello, world!"))
```

<Note>If you're running this inside a Lightning Studio with VSCode, make sure you're using the Python interpreter for your conda environment. The VSCode interpreter and your conda environment's interpreter may differ. Run the code from the terminal to avoid this.</Note>
