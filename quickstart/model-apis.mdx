Lightning AI hosts models you can query directly via API. Every account includes 30 million free tokens per month. You don't need a Studio to get started.

## Before you begin

Retrieve your API key from the Lightning AI dashboard.

## Option 1: litAI SDK

litAI is Lightning's Python SDK for model inference.

1. Install litAI.

   ```bash
   pip install litai
   ```

2. Set your API key as an environment variable or pass it directly in your code.
