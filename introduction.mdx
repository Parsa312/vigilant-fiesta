Use Firecrawl to get web data into a accessible format for AI agents and LLMs. You can use the Firecrawl API to:

- Search the web
- Scrape any page
- Interact with any page

 The API takes care of:

- Headless browser rendering, including JavaScript-rendered pages that a plain HTTP request would return empty
- HTML parsing
- Rate-limit handling

In return, you get clean data back in the following formats:

- LLM-ready Markdown
- Structured JSON
- Screenshots

## Core concepts

- **Scrape** — fetch a single URL and get back Markdown, HTML, links, structured data, or a screenshot.
- **Crawl** — start from a URL, follow links across a site, and scrape every page that matches your filters. Crawls run asynchronously; you poll a status endpoint until they finish.

Both endpoints accept the same `formats` and extraction options, so what you learn scraping one page applies directly to crawling a whole site.
