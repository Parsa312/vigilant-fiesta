Firecrawl is a web data API built for AI agents and LLMs. Instead of wiring up headless browsers, HTML parsers, and rate-limit handling yourself, you send a URL (or a crawl target) to Firecrawl and get back clean, LLM-ready Markdown, structured JSON, or screenshots — including for JavaScript-rendered pages that a plain HTTP request would return empty.

## Core concepts

- **Scrape** — fetch a single URL and get back Markdown, HTML, links, structured data, or a screenshot.
- **Crawl** — start from a URL, follow links across a site, and scrape every page that matches your filters. Crawls run asynchronously; you poll a status endpoint until they finish.

Both endpoints accept the same `formats` and extraction options, so what you learn scraping one page applies directly to crawling a whole site.
