Lightning AI gives you several ways to start building. Most users begin with a Studio — a persistent cloud environment where you can write code, run models, and deploy services. If you want to call a model without setting up an environment, you can use the model APIs directly. If you prefer notebooks, JupyterLab is available in the browser from any Studio.

  </Card>
  <Card title="Use model APIs" icon="code" href="/quickstart/model-apis">
