This guide gets you from zero to your first scrape. Every example is shown in cURL, Node, and Python, so you can follow along with whichever fits your stack.

## 1. Get an API key and store it as an environment variable

Sign up at [firecrawl.dev](https://www.firecrawl.dev) and grab an API key from your dashboard. Keys are prefixed `fc-`.

Store it as an environment variable rather than pasting it into code:

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

This only lasts for your current terminal session. To make it persist across new sessions, add it to your shell's startup file:

```bash
# bash
echo 'export FIRECRAWL_API_KEY="fc-your-api-key"' >> ~/.bashrc && source ~/.bashrc

# zsh (default on macOS since Catalina)
echo 'export FIRECRAWL_API_KEY="fc-your-api-key"' >> ~/.zshrc && source ~/.zshrc
```


## 2. Install the SDK

Install either the Python or Node JS SDK.

<CodeGroup>

```bash Python
pip install firecrawl-py
```

```bash Node
npm install firecrawl
```

</CodeGroup>

## 3. Scrape a website

<CodeGroup>

```python Python
from firecrawl import Firecrawl

app = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

result = app.scrape("https://firecrawl.dev")
print(result.markdown)
```

```javascript Node
import Firecrawl from "firecrawl";

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

const result = await app.scrape("https://firecrawl.dev");
console.log(result.markdown);
```

```bash curl
curl -X POST https://api.firecrawl.dev/v2/scrape \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://firecrawl.dev",
    "formats": ["markdown"]
  }'
```

</CodeGroup>

## 4. See the Markdown output

Firecrawl renders the page (including any JavaScript) and returns clean Markdown alongside metadata:

```json
{
  "success": true,
  "data": {
    "markdown": "# Power AI agents with clean web data\n\nThe complete toolkit to search, scrape, and interact with the web at scale...",
    "metadata": {
      "title": "Firecrawl - The context API to search, scrape, and interact with the web at scale.",
      "description": "Firecrawl is the context API to search, scrape, and interact with the web at scale...",
      "sourceURL": "https://firecrawl.dev",
      "statusCode": 200
    }
  }
}
```

<Note>
  The `markdown` and `metadata` values above are trimmed for readability; a real response includes the full page content. This example was captured from a live call to `/v2/scrape` while writing this doc.
</Note>
