The Lightning AI Hub lets you deploy dedicated models on any cloud in one click. Deployed models automatically scale up on demand and scale to zero when idle.

[Browse all models](https://lightning.ai/ai-hub)

## Deploy a model

1. Find a model on the [AI Hub](https://lightning.ai/ai-hub).
2. Select the model to open its detail page.
3. Choose the cloud account you want to run on.
4. Select **Deploy**.

![Deploying a prebuilt model from the AI Hub](https://storage.googleapis.com/lightning-avatars/litpages/01jetvsyq67be4rv3spekcn0dz/5486f18c-4c15-420a-a3f4-8a68402af768.webp)

Many models on the Hub are complete AI apps that are API-only — no web UI. This makes it straightforward to deploy scalable, production-ready AI without writing any code.

Contact [support@lightning.ai](mailto:support@lightning.ai) to learn more.

## Benefits

**Reduce costs** — serverless architecture ensures deployments only run when needed, eliminating unnecessary expenses.

**Autoscale on demand** — automatically adjusts to traffic spikes by scaling replicas without over-provisioning.

**Built-in observability** — logs, latency tracking, and usage metrics are included. No extra tools required.

**Versioning** — manage and update deployment versions without downtime, with safe rollbacks when needed.

**Multi-cloud support** — deploy across multiple cloud providers to avoid vendor lock-in.
