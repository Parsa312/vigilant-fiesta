This guide covers installing `firecrawl` and using it to scrape and crawl pages. Before you start, you'll need a Firecrawl API key stored as the `FIRECRAWL_API_KEY` environment variable. For more information, see [Quickstart](/quickstart) if you haven't set one up yet.


## Install

Install the `firecrawl` package using npm:

```bash
npm install firecrawl
```

## Authenticate

Create a `Firecrawl` client with your API key:

```javascript
import Firecrawl from "firecrawl";

const app = new Firecrawl({ apiKey: "fc-your-api-key" });
```

You can also read the key from an environment variable instead of hardcoding it:

```javascript
import Firecrawl from "firecrawl";

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
```

## Scrape a page

Call `scrape()` with a URL and the formats you want back:

```javascript
const result = await app.scrape("https://firecrawl.dev", {
  formats: ["markdown", "links"],
  onlyMainContent: true,
});

console.log(result.markdown);
console.log(result.metadata.title);
console.log(result.links.slice(0, 5));
```

`scrape()` is asynchronous — it awaits the rendered page and returns the requested formats directly, with the same fields documented in the [`/v2/scrape` reference](/api-reference/scrape).

## Crawl a site

Call `crawl()` with a starting URL to scrape it and its linked pages:

```javascript
const job = await app.crawl("https://docs.firecrawl.dev", {
  limit: 50,
  scrapeOptions: { formats: ["markdown"] },
});

console.log(job.status);              // "completed"
console.log(`${job.completed}/${job.total}`);

for (const page of job.data) {
  console.log(page.metadata.sourceURL, "->", page.markdown.length, "chars");
}
```
